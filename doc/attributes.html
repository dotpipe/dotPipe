Attributes

  • insert: Specifies the target element for content insertion.
  • ajax: Defines the URL for AJAX requests.
  • query: Sets query parameters for AJAX requests.
  • modal: Specifies an Irondocks file for templating.
  • download: Enables file downloads.
  • x-toggle: Toggles classes on specified elements.
  • clear-node: Clears content of specified nodes.
  • redirect: Follows AJAX calls in POST or GET mode.
  • event: Specifies events for dynamic elements.
